Twikoo Server

Status: Running

API Endpoint: /api/twikko

About

This is a Twikoo server instance running on Vercel. Twikoo is a simple, safe, and free comment system.

Usage

To use this server with your Twikoo client, configure your client to point to:

https://[your-domain]/api/twikko

Documentation

For more information about Twikoo, visit the official documentation.
